# Taskingen

Run npm and shell scripts from the Explorer sidebar in VS Code / Cursor.

## Install (local)

1. Build a VSIX: `npm run package` → creates `taskingen-<version>.vsix`
2. Command Palette → **Extensions: Install from VSIX…** → select that file

## Usage

- Open the **Explorer** sidebar; **Taskingen** appears below the file tree.
- Drag the view to the primary/secondary sidebar or panel if you prefer.
- Use **Reset Location** to return it to Explorer.
